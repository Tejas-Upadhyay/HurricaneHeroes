WELCOME TO YOUR RELIEF MANAGEMENT SYSTEM!

This is a complete design implementation with static data.
No database connectivity is included - perfect for UI demonstration.

================================================================================
                    QUICK START INSTRUCTIONS
================================================================================

STEP 1: Install Dependencies
    Open terminal/command prompt and run:
    > pip install -r requirements.txt

STEP 2: Run the Server
    > python manage.py runserver

    OR double-click: run.bat (Windows)
    OR run: ./run.sh (Linux/Mac)

STEP 3: Open Browser
    Navigate to: http://127.0.0.1:8000/

================================================================================
                      USER INTERFACES
================================================================================

1. PUBLIC PANEL (No Login Required)
   - Home: http://127.0.0.1:8000/
   - Areas: http://127.0.0.1:8000/areas/
   - Features: View statistics, browse areas, see relief needs

2. AREA ADMIN PANEL
   - Login URL: http://127.0.0.1:8000/login/
   - Use any username/password
   - Select "Area Admin" role
   - Access dashboard, manage needs, view products

3. SUPER ADMIN PANEL
   - Login URL: http://127.0.0.1:8000/login/
   - Use any username/password
   - Select "Super Admin" role
   - Access full system overview, manage areas and admins

================================================================================
                      DEMO CREDENTIALS
================================================================================

Username: admin (or any)
Password: admin (or any)

Role: Select from dropdown
  - Super Admin (for full access)
  - Area Admin (for area-specific access)
