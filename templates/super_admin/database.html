Instructions
Export Database:
  1. Click the "Export Database Backup" button
  2. The backup file will be downloaded automatically
  3. Save the file in a safe location
  4. The file name will include a timestamp (e.g., backup_20251210_011944.sql)
Import Database:
  1. Click "Choose File" and select your SQL backup file
  2. Click "Import Database" button
  3. Confirm the import action
  4. A safety backup of your current database will be created automatically
  5. Your database will be restored from the selected file
Tip: Always export your database before making major changes or importing new data. This way you can restore if something goes wrong.
